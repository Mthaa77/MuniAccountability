# MuniAccountability Command

Premium municipal oversight and recovery operating system for South African institutional users.

## Local Development

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:3000`.

## Current Prototype Scope

- Executive Command Centre for a one-province Provincial Treasury pilot.
- AGSA-first intervention queue, Municipality 360 case file, action workflow, briefing builder, and source/data operations panels.
- Public MuniCheck and MuniData entry points represented in the shell.
- Versioned `/v1/*` API family backed by typed pilot data.

The prototype deliberately marks Treasury/Municipal Money telemetry as pending validation. It does not claim live financial integration.

## Verification Commands

```bash
npm run typecheck
npm run build
```
