# Prototype data notice

MuniAccountability Command is currently an AGSA-backed prototype workspace.

## Safe demo statements

- The platform shows structured AGSA report extracts.
- The platform shows page citations, confidence flags and review states.
- The platform models municipal oversight workflows.
- The platform separates public-safe profiles from institutional workflow data.
- The platform shows which production gates still need evidence.

## Boundaries

- Treasury or Municipal Money integration is gated.
- Platform risk scores are prioritisation aids.
- Exact municipality-level audit outcomes need annexure mapping where unresolved.
- MuniCheck should remain public-safe.
- Production gates need supporting evidence before promotion.

## External demo wording

This is a prototype municipal oversight platform using structured AGSA-backed source data and review workflows. It supports evidence-based oversight and does not replace official audit reports, municipal records or Treasury systems.
